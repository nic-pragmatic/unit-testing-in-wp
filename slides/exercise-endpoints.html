Exercise: Creating Custom Endpoints

☛   Create a CPT and make it available to the app

  • Create a custom wp/v2/ideas endpoint

    This should use out-of-the-box WordPress custom post type functionality.

  • Add custom fields reminder_date & reminder_content

    To avoid having them be buried within the 'meta' subkey, we want to use register_rest_field() functionality.

  • [ Bonus ] - Use granular permissions

    Create a custom role and corresponding capabilities to only let users of the idea_manager role manage ideas.
