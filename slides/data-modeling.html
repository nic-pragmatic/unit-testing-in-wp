Data Modeling

    Logic is powered by and runs on data

  • Client/server architecture

    Data needs to be modeled so it serves the purposes of both the server as well as the client context

  • REST resources

    For a proper REST API usage, data should be grouped into resources that fit the HTTP protocol structure
