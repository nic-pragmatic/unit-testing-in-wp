Setting up your environment

  • Local WordPress site

    Use your favorite environment to set up an empty site.

  • Node.js and npm

    Go to nodejs.org and follow instructions (platform-specific).

  • Ionic

    Run the command npm install -g ionic.

  • Fetch app source code

    git clone https://github.com/schlessera/wcbtn-2018-app

  • Compile app requirements

    Run npm install from within the wcbtn-2018-app folder
