Connecting to the App

WordPress

  • Navigate to wp-config.php and add a JWT Secret Key

    
					define(
					  'JWT_AUTH_SECRET_KEY',
					  'Oo,I7#nl{|*,8uhGGfU4z@fR&$uBjab54qCZ:rd3|6'
					);

Ionic

  • Navigate to src/environment.ts in your ionic project folder

    
				export const environment = {
				  site_url  : 'YOUR_LOCAL_INSTALL_URL',
				  ideas_url : '/wp-json/wp/v2/ideas',
				  jwt_url   : '/wp-json/jwt-auth/v1/token',
				}

Terminal

  • Navigate to your ionic project folder

    
				$ ionic serve
